# Les 1 - Python

## 1.1 Inleiding

Python is veel meer dan een programmeertaal. Het is een praktische tool die door programmeurs, data-analisten, onderzoekers en zelfs financiële professionals wordt gebruikt. In deze les maakt u op een toegankelijke manier kennis met de basisprincipes van Python. U leert hoe u de taal installeert op uw computer, hoe u code schrijft en uitvoert, en hoe u de interactieve shell gebruikt als oefenomgeving.

Daarnaast maakt u kennis met typische eigenschappen van Python, zoals het gebruik van witruimte om structuur aan te brengen in uw code. Deze aanpak bevordert de leesbaarheid en voorkomt veelgemaakte fouten. U ontdekt hoe u gegevens invoert, hoe u eenvoudige berekeningen uitvoert en hoe u programma’s opbouwt met behulp van lussen en voorwaardelijke logica.

Naast de technische kant, wordt ook de cultuur rond Python belicht. Met de ‘Zen van Python’ als leidraad leert u hoe eenvoud, duidelijkheid en consistentie bijdragen aan goed programmeerwerk. U ervaart hoe een duidelijke schrijfstijl niet alleen uzelf, maar ook andere programmeurs helpt.

Deze les vormt het startpunt van een leertraject waarin u stap voor stap vertrouwd raakt met een taal die wereldwijd wordt ingezet en waarin ook u vaardig kunt worden.

### 1.4.2 Invoerfunctie

Python heeft veel ingebouwde functies, waaronder een invoerfunctie.

Met de invoerfunctie kunt u een gebruiker om tekstinvoer vragen. Deze functie laat het programma stoppen en wachten tot de gebruiker de gegevens heeft ingetoetst. Python 2 heeft de ingebouwde functie `raw_input()`, bij Python 3 is dat `input()`. Het programma wordt in beide gevallen hervat zodra de gebruiker op Enter drukt.

Om de invoer in uw programma vast te leggen, heeft u een **variabele** nodig. Zie een variabele als een doos waar u een waarde kan in stoppen. In een latere les leert u hierover meer.

U kunt de invoer nemen en toekennen aan een variabele. Dit doet u door de operator `=` voor het trefwoord en de naam van de variabele voor de operator te zetten. Een voorbeeld:

*Wanneer u wilt dat de gebruiker een waarde aan uw programma doorgeeft, kunt u gemakkelijk gebruikmaken van de*`input()`*-functie. Het is altijd nodig dat uw programma een variabele bevat waarin de invoer van de gebruiker opgeslagen wordt. U maakt bijvoorbeeld de variabele*`my_var`*aan, die gelijk staat aan de invoer van de gebruiker. U kunt vervolgens de waarde afdrukken die is opgeslagen in*`my_var`*. In het volgende voorbeeld noemen we de variabele*`leuke_naam`*:*

```python
>>> leuke_naam = input()
# gebruiker typt de naam 'Linda'
>>> print(leuke_naam)
# programma toont de ingevoerde naam 'Linda'
```
