18.11 Bijlage
Mediaquery’s behorende bij algemene breekpunten

| Apparaatgrootte | Mediaquery |
|-----------------|------------|
| Hele kleine apparaten (< 600 pixels, zoals smartphones) | @media only screen and (max-width: 600px) |
| Kleine apparaten (600-767 pixels, zoals grote smartphones) | @media only screen and (min-width: 600px) |
| Middelgrote apparaten (768-991 pixels, zoals tablets) | @media only screen and (min-width: 768px) |
| Grote apparaten (992-1199 pixels, zoals laptops/desktopcomputers) | @media only screen and (min-width: 992px) |
| Hele grote apparaten (> 1200 pixels) | @media only screen and (min-width: 1200px) |
